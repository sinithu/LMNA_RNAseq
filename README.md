# RNAseq analysis

This repository contains R scripts used for [differential gene expression](https://github.com/sinithu/LMNA_RNAseq/tree/main/DGE_analysis.R) and [overrepresentation](https://github.com/sinithu/LMNA_RNAseq/tree/main/ORA.R) analyses in the manuscript "Decreased mechanosensing properties and altered gene expression in hiPSC-cardiomyocytes carrying a lamin A/C mutation", currently in preparation.
